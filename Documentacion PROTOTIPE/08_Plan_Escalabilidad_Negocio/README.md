# 📁 09 — Plan de Escalabilidad del Negocio

Esta carpeta es el **centro estratégico y de visión de largo plazo** del ecosistema Prototipe. Aquí se guardan los planes de negocio, hojas de ruta verticales y decisiones estratégicas que guían el crecimiento del Ecosistema.

> ⚠️ **Solo el chat central de App Ventas tiene permiso de escritura en esta carpeta.** Los chats de cliente son de solo lectura.

---

## Contenido

| Archivo | Descripción |
|---|---|
| [hoja_de_ruta_maestro.md](./hoja_de_ruta_maestro.md) | Visión global del negocio: nichos, fases de crecimiento, apps base planificadas y backlog estratégico. |

---

## Cómo usar esta carpeta

- Cuando se tome una **decisión estratégica de negocio**, regístrala en `hoja_de_ruta_maestro.md`.
- Si un nicho madura lo suficiente para tener su propio plan técnico detallado, crea un archivo separado (ej: `vertical_restaurante.md`).
- Cada vez que se alcance una fase, marca las tareas como completadas `[x]` en el documento correspondiente.
