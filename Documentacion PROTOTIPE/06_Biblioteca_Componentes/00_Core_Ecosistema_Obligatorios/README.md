# Componentes Core Obligatorios del Ecosistema

Esta categoría agrupa todos los componentes que son de carácter **MANDATORIO** y deben estar presentes de forma idéntica en todas las aplicaciones e instancias de clientes aprovisionadas bajo el ecosistema **PROTOTIPE**. 

El propósito de esta suite es garantizar la consistencia en la facturación comisional, telemetría técnica, soporte al desarrollador y auditorías legales de servicio.
