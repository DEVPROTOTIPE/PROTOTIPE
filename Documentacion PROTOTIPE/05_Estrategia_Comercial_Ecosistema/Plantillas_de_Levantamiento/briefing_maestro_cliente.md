## OBJETIVO

Este documento tiene como propósito recopilar toda la información necesaria para comprender un negocio, identificar oportunidades de mejora, detectar procesos susceptibles de automatización y diseñar una solución tecnológica alineada con sus necesidades.

La información recopilada servirá como base para:

- Diagnóstico operativo.
- Propuesta técnica.
- Propuesta comercial.
- Estimación de esfuerzo.
- Definición de arquitectura.
- Parametrización del sistema.
