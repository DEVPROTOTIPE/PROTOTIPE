# 🧠 Mapa Semántico para IA — PRUEBACCC345310

Referencia directa de archivos clave para que la IA navegue el proyecto sin búsquedas ciegas.

> Ejecutar `npm run map` para regenerar este mapa con la estructura actual.

## Archivos Críticos
| Archivo | Propósito |
|---|---|
| `src/App.jsx` | Entrada principal, enrutador y providers |
| `src/store/` | Stores de estado global (Zustand) |
| `src/hooks/` | Hooks personalizados de lógica de negocio |
| `.env.local` | Variables de entorno — NO editar en código |
